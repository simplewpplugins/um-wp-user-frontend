Allows adding WP User Frontend's post form into Ultimate member's profile tab.

== Description ==

*WP User Frontend Integration for Ultimate Member* plugin requires both [Ultimate Member](https://wordpress.org/plugins/ultimate-member) and [WP User Frontend](https://wordpress.org/plugins/wp-user-frontend) plugin to work.

This plugin adds an option for you to add Profile tab with frontend post form on Ultimate member's user profile page.

== Installation ==
**This section describes how to install the plugin and get it working**
= AUTOMATIC INSTALLATION (EASIEST WAY) =

To do an automatic install of *WP User Frontend Integration for Ultimate Member*, log in to your WordPress dashboard, navigate to the Plugins menu and click
Add New.
In the search field type *"WP User Frontend Integration for Ultimate Member"* by Simple Plugins. Once you have found it you can install it by simply clicking
"Install Now" and then "Activate".

= MANUAL INSTALLATION =

**Uploading in WordPress Dashboard**

* Download um-wp-user-frontend.zip
* Navigate to the *'Add New'* in the plugins dashboard
* Navigate to the *'Upload'* area
* Select *um-wp-user-frontend.zip* from your computer
* Click 'Install Now'
* Activate the plugin in the Plugin dashboard

**Using FTP**

* Download *um-wp-user-frontend.zip*
* Extract the *um-wp-user-frontend* directory to your computer
* Upload the *um-wp-user-frontend* directory to the /wp-content/plugins/ directory
* Activate the plugin in the Plugin dashboard

The WordPress codex contains <a href="https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation">instructions</a> on how to install a WordPress plugin.
